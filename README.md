# Expense-management-system-java-SQlLite
Expense Management System (JavaFX + SQLite)
This project is a desktop-based Expense Management System developed using Java, JavaFX, SQLite, and Maven. The application is designed to help users efficiently manage their daily expenses by providing a simple and interactive graphical user interface for recording, updating, viewing, and maintaining financial transactions.
The primary objective of this project was to gain practical experience in desktop application development, database integration, and software design while building a useful real-world solution. The application allows users to store expense information in a structured SQLite database, ensuring reliable data storage and retrieval. Through the implementation of CRUD (Create, Read, Update, Delete) operations, users can easily manage expense records and keep their financial information organized.
The system utilizes JavaFX to create a responsive and user-friendly interface that improves usability and navigation. SQLite was selected as the database management system due to its lightweight architecture and seamless integration with Java applications. Maven was used for project management and dependency handling, ensuring a well-structured and maintainable codebase.
Key Features
Add new expense records
View and manage existing expenses
Update expense details when required
